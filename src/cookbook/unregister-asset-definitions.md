# How to Unregister an Asset Definition

```rust
fn undefine_asset(
    iroha: &Client,
) {
    let hats = "hat#outfit".parse::<AssetDefinitionId>().unwrap();
    iroha.submit(Unregister::asset_definition(hats)).unwrap();
}
```
